```java
public class Hello{
  public static void main(String[] args){
    System.out.println("Hello");
  }
}
```

1.`public class Hello`

​	Hello 라는 클래스를 선언한다.



2.`public static void main(String[] args) { ... }`

​	프로그래밍을 실행시키는 메서드.

​	실행 시작 위치.



3.`System.out.println("...");`

​	" "(큰 따옴표) 사이의 내용을 출력하는 기능.

​	println( )은 출력 후 다음 줄로 이동한다.
